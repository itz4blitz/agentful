## Summary

These diagrams illustrate:

1. **Extension Structure**: How agentful Studio fits into VS Code's UI
2. **Authentication Flow**: Complete OAuth flow with browser redirect
3. **Workspace Opening**: User-initiated panel opening (not automatic)
4. **Internal Architecture**: How extension components communicate
