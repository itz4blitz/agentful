- **Capabilities:** Checkboxes with counts

## Responsive Design

### Breakpoints
```css
/* Mobile (sidebar collapsed) */
@media (max-width: 640px) {
  .sidebar { display: none; }
  .main-panel { grid-template-columns: 1fr; }
}

/* Tablet (sidebar icons only) */
@media (min-width: 641px) and (max-width: 1024px) {
  .sidebar { width: 60px; }
  .sidebar-text { display: none; }
}

/* Desktop (full sidebar) */
@media (min-width: 1025px) {
  .sidebar { width: 280px; }
  .main-panel { grid-template-columns: 280px 1fr; }
}
```

### Touch Targets
- **Minimum:** 44x44px (iOS standard)
- **Recommended:** 48x48px
- **Spacing:** 8px between targets

### Font Sizes
- **Headings:** 16px - 24px
- **Body:** 14px (desktop), 16px (mobile)
- **Small:** 12px
- **Code:** 13px monospace
