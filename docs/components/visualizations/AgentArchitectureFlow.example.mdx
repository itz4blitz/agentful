## Features

- **Interactive Nodes**: Click any agent to view detailed information
- **Hexagonal Layout**: 7 specialist agents arranged around central orchestrator
- **Bidirectional Edges**: Animated connections showing delegation flow
- **Tooltips**: Rich tooltips with agent skills and descriptions
- **Zoom & Pan**: Full viewport controls for exploration
- **Mini-map**: Bottom-right navigation helper
- **Accessible**: ARIA labels and keyboard navigation support
- **Responsive**: Adapts to container size
- **Glassmorphism**: Modern blur effects and emerald green theme
- **Animations**: Pulse effect on orchestrator, hover effects on all nodes

## Agent Details

Each agent node displays:
- Icon and name
- Color-coded connections
- Skills and capabilities (on click)
- Animated hover effects
- Glassmorphism styling

The central **Orchestrator** coordinates all specialist agents through bidirectional communication channels.
