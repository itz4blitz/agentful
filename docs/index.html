How It Works

Four simple steps to autonomous development

Initialize

Run npx @itz4blitz/agentful init in your project

2

Define Product

Edit PRODUCT.md or .claude/product/index.md with your requirements

3

Start Claude

Run claude and type /agentful-start

4

Watch It Build

agentful works autonomously until complete
