## Quick Installation

**Prerequisites:**
- [Claude Code](https://code.anthropic.com)
- Node.js 22+

**Install in your project:**

```bash
npx @itz4blitz/agentful init
```

This creates:
- `.claude/` - Agents, commands, skills (version control this)
- `.agentful/` - Runtime state (gitignored)
- `.claude/product/` - Product specification structure

## What is agentful?

**agentful** is a self-hosted AI agent server that runs Claude Code agents on your infrastructure:

- **Self-Hosted Server** - Deploy with `agentful serve` on your own infrastructure (Oracle Cloud free tier, AWS, GCP, on-prem)
- **Multi-Platform CI/CD** - Native support for GitHub Actions, GitLab CI, Jenkins, CircleCI, Bitbucket Pipelines
- **Auto-Generated Agents** - Architect analyzes your stack and generates specialized agents for your frameworks
- **Complete Agent Toolkit** - 8 core agents, 6 extensible skills, custom hooks, adaptive quality gates
- **Human-in-the-Loop** - You control decisions while agents handle implementation

## After Installation

Once agentful is installed:

1. **Define your product**: Edit `.claude/product/index.md` to describe what you're building
2. **Start Claude Code**: Run `claude` in your project directory
3. **Begin development**: Type `/agentful-start` to start structured development
4. **Check progress**: Use `/agentful-status` to see completion percentage

For more details, see the [Quick Start Guide](/getting-started/quick-start).

## Advanced Installation Options

For custom installations, see the [Installation Guide](/getting-started/installation) to configure:

- **Presets** - Minimal, full-stack, backend-only configurations
- **Agents** - Choose which specialized agents to install
- **Skills** - Add portable expertise modules
- **Hooks** - Session start commands and health checks
- **Quality Gates** - Configure automated validation rules
- **Tech Stack** - Explicitly specify your technology stack
