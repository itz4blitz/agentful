## What is agentful?

AI development toolkit that coordinates specialized agents in Claude Code to build software from product specs.

**Key capabilities:**
- **[Parallel execution](/concepts/background-agents)** - Multiple agents work simultaneously in isolated git worktrees (frontend + backend + tests running concurrently)
- **[Three-tier architecture](/getting-started/overview#agent-system)** - Core (pre-built), domain (generated for your stack), ephemeral (temporary, task-specific)
- **[Shared skills](/skills/validation)** - Reusable capabilities like validation, testing, and research across all agents
- **[Custom hooks](/hooks)** - Intercept and modify agent behavior at key lifecycle points
- **[Quality gates](/concepts/quality-gates)** - Automated validation for every change (tests, types, coverage, security)
- **Human checkpoints** - You decide, agents execute

## Our Philosophy

Every software project has three tiers of agents and skills:

1. **Core** (included) - Universal agents like orchestrator, reviewer, fixer + shared skills like validation, testing
2. **Domain** (generated) - Custom agents for your stack (backend, frontend) + project-specific skills
3. **Ephemeral** (temporary) - Short-term agents/skills for specific tasks that get deleted after use

**Skills are shared across agents** - any agent can use validation, testing, research, etc. This gives you ready-to-use components out-of-the-box while adapting to your project's unique patterns. For custom workflows, start with just the orchestrator and add only what you need.
