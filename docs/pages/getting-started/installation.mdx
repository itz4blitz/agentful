# Installation

## Prerequisites

- Claude Code - [Install here](https://code.anthropic.com)
- Node.js 22+
- A project directory

## Install

```bash
cd your-project
npx @itz4blitz/agentful init
```

This analyzes your project, detects your tech stack, and creates:
- `.claude/` - 7 agents, 5 commands, skills
- `.agentful/` - Runtime state (gitignored)
- Either `PRODUCT.md` (simple) or `.claude/product/` (complex)
- `CLAUDE.md` - Project instructions

## What Gets Created

```
your-project/
├── PRODUCT.md              # Product spec (simple projects)
├── CLAUDE.md               # Project instructions
├── .claude/
│   ├── agents/             # 7 specialized agents
│   ├── commands/           # 5 slash commands
│   ├── skills/             # Domain skills
│   ├── product/            # Product spec (complex projects)
│   │   ├── index.md
│   │   └── domains/
│   └── settings.json
└── .agentful/              # Runtime state (gitignored)
    ├── state.json
    ├── completion.json
    ├── decisions.json
    └── architecture.json
```

## Structure Detection

agentful auto-detects the best structure:

**Simple projects** (< 3 domains, single framework):
- Creates `PRODUCT.md` at root
- Single file, easier to start

**Complex projects** (≥ 3 domains, multiple frameworks, monorepo):
- Creates `.claude/product/index.md` with `domains/`
- Organized by domain
- Preferred for most projects

You can switch between structures anytime. agentful auto-detects on each run.

## Verify Installation

```bash
# Check directories exist
ls -la .claude/
ls -la .agentful/

# Check state
cat .agentful/state.json

# Check structure
ls -la PRODUCT.md              # Simple
ls -la .claude/product/        # Complex
```

## Init Options

### Bare Init

Skip template files if you already have your spec:

```bash
npx @itz4blitz/agentful init --bare
```

### Skip Smart Analysis

Faster init without project analysis:

```bash
npx @itz4blitz/agentful init --no-smart
```

### Force Hierarchical

Force hierarchical structure regardless of complexity:

```bash
npx @itz4blitz/agentful init --generate-domains
```

### Global Install

For frequent use:

```bash
npm install -g @itz4blitz/agentful
agentful init
```

Note: `npx` always uses the latest version.

## Troubleshooting

### Command not found: npx

Install Node.js 22+ from [nodejs.org](https://nodejs.org/)

```bash
node --version  # Should be v22+
npx --version
```

### .claude directory already exists

Backup before overwriting:

```bash
mv .claude .claude.backup
npx @itz4blitz/agentful init
```

### Permission denied

```bash
ls -la
sudo chown -R $USER:$USER .claude .agentful
```

### Network timeout

```bash
# Clear cache
npm cache clean --force
npx @itz4blitz/agentful init
```

## Uninstall

```bash
rm -rf .claude .agentful PRODUCT.md CLAUDE.md
```
