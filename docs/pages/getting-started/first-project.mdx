# Your First Project

Build a task management app from scratch.

## Project Overview

**What we're building**:
- Add tasks
- Mark tasks complete
- Delete tasks
- Filter by status

**Tech Stack**:
- Next.js 14 + TypeScript
- Tailwind CSS
- Local state (no database)
- Vitest for testing

**Time**: 30-45 minutes

## Step 1: Initialize Project

```bash
mkdir task-manager
cd task-manager
npx create-next-app@latest . --typescript --tailwind --eslint --app --src-dir --import-alias "@/*"
npx @itz4blitz/agentful init
```

agentful initializes your project structure:
1. Creates `.claude/` directory
2. Generates product template at `.claude/product/index.md`
3. Sets up core agents (orchestrator, backend, frontend, tester, reviewer, fixer)
4. Configures hooks and permissions

Now start Claude Code:

```bash
claude
```

You'll see the welcome message:

## Common Issues

### agentful stops mid-development

Check status:
```bash
/agentful-status
```

Look for:
- Pending decisions â†’ `/agentful-decide`
- High iteration count â†’ `/agentful-generate` to resume
- Errors â†’ `/agentful-validate`

### Tests failing

Let agentful handle it. Fixer agent will:
1. Identify failing tests
2. Fix implementation
3. Re-run tests
4. Continue when passing

### TypeScript errors

Fixer agent resolves these. If persistent:
```bash
npx tsc --noEmit
# Fix manually if needed
/agentful-generate  # Continue
```

## Customizing Your Project

### Add New Features

Edit `.claude/product/index.md`:
```markdown
### 6. Task Due Dates - MEDIUM
**Description**: Add due dates to tasks

**Acceptance Criteria**:
- [ ] Date picker on add task
- [ ] Display due date on task
- [ ] Sort by due date
```

Then:
```bash
/agentful-generate
```

### Modify Existing Features

Update acceptance criteria in `.claude/product/index.md`, then run `/agentful-generate`.

### Add Domain Structure

For complex projects, organize into domains:

```bash
mkdir -p .claude/product/domains/tasks
mv .claude/product/index.md .claude/product/index.md.bak
# Split content into domain files
```
