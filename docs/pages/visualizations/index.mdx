## Integration with agentful

These visualizations are not just documentation - they reflect the actual state of your agentful project:

**Agent Architecture** ‚Üí Reads `.agentful/architecture.json` to show your custom agent system

**Product Structure** ‚Üí Parses `.claude/product/` to visualize your product specification

**Workflow Pipeline** ‚Üí Reads `.agentful/state.json` to show current development phase

**Quality Gates** ‚Üí Displays data from `.agentful/last-validation.json`

Run `/agentful-status` in Claude Code to generate the state files these visualizations consume.

## Usage Examples

### In Documentation Sites

Import and use any visualization in your MDX documentation:

```mdx
import { AgentArchitectureFlow } from '../components/visualizations/AgentArchitectureFlow'

<AgentArchitectureFlow />
```

### In React Applications

Use as regular React components:

```tsx
import { QualityGateDashboard } from './components/visualizations/QualityGateDashboard'

function App() {
  return (
    <div>
      <h1>Quality Status</h1>
      <QualityGateDashboard
        gates={myGates}
        showOverallScore={true}
      />
    </div>
  )
}
```

### In CLI Output

Generate static SVG exports for terminal display:

```bash
# Generate visualization snapshot
/agentful-status --export-svg=architecture.svg
```
