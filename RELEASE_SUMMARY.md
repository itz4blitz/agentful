#### Auto-Detection

**Agentful automatically detects which format you're using** - no configuration needed!

**Detection Priority**:
1. Check for `.claude/product/domains/*/index.md` ‚Üí Hierarchical
2. Check for `PRODUCT.md` at root ‚Üí Flat
3. Check for `.claude/product/index.md` ‚Üí Flat (new location)
4. Error if none found

**Migration Path**:
Start with flat, migrate to hierarchical when you outgrow it:

```bash
# 1. Create domain directories
mkdir -p .claude/product/domains/{auth,tasks,billing}

# 2. Move features to domain files
# 3. Update completion.json format
# 4. Done! Agentful auto-detects the change
```
